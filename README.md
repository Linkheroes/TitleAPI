# TitleAPI
# Require Spigot 1.8

Pour send un Title via l'api écrivez la ligne suivant :

Title.sendTitle(Player, String, String, int);

Pour send une actionBar écrivez la ligne suivant :

Title.sendActionBar(Player, String);
